# ⚡ Zoë

> **"Stay sharp, Operator. The system is always watching."**

**Zoë** is a hardcore, gamified productivity and study-tracking application built with a classic retro-cyber aesthetic. It transforms studying and task management into a high-stakes role-playing experience, enforcing absolute discipline through a unique psychological economy. 

Operators must earn XP, manage vital health metrics, survive global "System Audits," and manage a deflationary digital currency to buy critical survival gear or access volatile black-market overclocks.

---

## 🎮 The Economy Philosophy

Zoë is engineered around the core principles of **Digital Scarcity** and **Loss Aversion**:
* **Controlled Inflation:** Earning currency is a ruthless grind.
* **Mandatory Survival:** Requisitions store items are essential to defend your profile against recurring automated system decay.
* **High Risk, High Reward:** The Black Market forces fierce competition for hyper-limited stock, rewarding swift and decisive action.

---

## 🚀 Key Features

### 📋 High-Stakes Task Management
Add, track, and execute your operations. Be warned: failing to maintain structural momentum or missing milestones leads to direct health deductions.

### 🧬 Cyber-RPG Progression & Skill Trees
Gain XP to level up your security clearance, unlock system achievements, and navigate a complex skill tree packed with custom augments.

### 💰 Deflationary Requisitions & Black Market
Spend your hard-earned currency in the **Requisitions Shop** for Medkits and Titanium Shields to survive, or slip into the **Black Market** for volatile, illegal, temporary system overclocks.

### 🚨 Real-Time System Audits & Global Events
Survive automated global audits that drain unshielded operator health. Administrators have the power to broadcast real-time server-wide Double XP modes and localized core item airdrops instantaneously to all active web clients.

### 🖥️ Admin Control Room
A high-clearance operator command dashboard allowing administrators to monitor the global player database, fine-tune economic values, and trigger live event protocols.

## 🛠️ Technology Stack

* **Frontend:** React.js (Vite) + Redux Toolkit (State Architecture)
* **Styling:** Vanilla CSS with a strict, flat, instant-snap retro-cyber interface layout.
* **Backend:** Serverless Node.js API (Vercel Functions)
* **Database:** MongoDB (Mongoose ORM)
* **Deployment:** Vercel (Edge Network Integration)
